### WP Headless Menu Editor

#### Menu editor for headless WordPress - independent of Appearance. Adds menus to WP REST API v2.

#### WP Headless Menu Editor endpoints:
* /wp-json/hme/v1/menus/
* /wp-json/hme/v1/menus/{id}
